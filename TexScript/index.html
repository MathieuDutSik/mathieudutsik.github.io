LatexScript

  • Sometimes, when submitting a paper we all realize we have a problem ...
  • ... the editors have very strict requirements in the format of the paper!
  • It can be very normal, like no uncited references, page formatting, no unneeded files, ...
  • ... but it can also be more intrusive like names of your figure files, name no subdirectory in your structure
The set of perl scripts available here is supposed to make your task slighly easier. They put some mild requirements on the latex style of your paper:
  • At most one image per line
  • inputs should be at the beginning of the line
Those programs
  • Can unfold a directory, subdirectory structure
  • Can copy only the files needed from one directory to another
  • Can check order of figures with order of reference
  • Can do renaming of filename of .eps file according to various standard
  • Can check for uncited reference
  • Can check for uncited equation
  • Can merge all \input{...}
  • Can create an arxiv archive
Last remarks:
  • From time to time, I find some bugs, this is somehow unavoidable ...
  • ... so, ALWAYS DO SAVING before using those programs.
  • Those programs do not substitute the reading of your file ThePaper.log, they complement it. Much is already there for you
