Root Matroid

The Root Matroid GAP package is designed for computing the automorphism groups of matroid root systems, a task critical in combinatorial representation theory and geometry.

Technical Overview

  • Computation Engine: Utilizes the external nauty program for efficient isomorphism and automorphism group calculations.
  • Specialized Implementation: Unlike the standard GRAPE implementation in GAP, this package is specifically optimized to handle vertex coloring, which is essential for matroid root systems.
