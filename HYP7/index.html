Extreme rays of HYP7

The EXT-HYP7 program is dedicated to the enumeration of orbits of extreme rays for the hypermetric cone on seven vertices (HYP7).

Software Requirements

To run the enumeration, the following software is required:

  • cdd: A library for generating all vertices of a convex polytope.
