Space Fullerenes

This page contains supplementary data for the research paper: M. Dutour Sikirić, M. Deza, Space fullerenes: computer search for new Frank-Kasper structures II, Structural Chemistry 23-4 (2012) 1103--1114.

Data Sets

The following archives contain structural data for the 84 space fullerene structures identified in the paper:

  • CGD Files (tar.gz): Crystallographic Group Data format.
  • DS Files (tar.gz): Delaney Symbol format.
