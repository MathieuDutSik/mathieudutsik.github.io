ChemicalReact

ChemicalReact is a computational tool designed to determine equilibrium concentrations of chemical species given their initial concentrations and reaction parameters.

Workflow

  1. Load Reaction Data: Import your reaction file (e.g., PhosphoricAcid_react).
  2. Define Concentrations: Import the initial concentration values (e.g., PhosphoricAcid_conc).
  3. Compute: Execute the solver to find the equilibrium state.
  4. Export: Save the resulting data for further analysis.
