Cube Tiling & Packing

This project focused on the classification of periodic cube tilings and packings in various dimensions. The classification for dimension 4 was a significant computational effort, requiring several months of processing time.

Programs & Scripts

  • CubeTiling.g: The core classification procedure implemented in GAP.
  • EnumerationDimN: A script for dimension-specific enumeration.
  • CreateHtmlFile: Utility for generating result pages.
