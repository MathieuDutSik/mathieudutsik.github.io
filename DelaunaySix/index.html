Six Dimensional Delaunay Polytopes

The DelaunaySix program provides a comprehensive enumeration of all combinatorial types of Delaunay polytopes in dimension six.

Software Requirements

To run the enumeration, the following software is required:

  • GAP (Groups, Algorithms, Programming)
  • nauty (Graph Isomorphism Tool)

Download

  • DelaunaySix Source (tar.gz)

Classification Results

The results are categorized by corank and rank. Each category links to Delaunay polytopes related to the Schläfli polytope and those related to cuts (metrics expressed as a sum of cuts).
