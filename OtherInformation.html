Programs and Data sets

UPDATE 2021: Most new development are now taking place on github in github:MathieuDutSik.

Programs for geometric applications

  • PlanGraph a package for handling planar graphs
  • Polyhedral a package for handling polytopes and lattices
  • Poset programs for working with posets (mostly petrie schemes)

Programs for oceanographic applications

  • PGMCL a simple set of fortran procedure for coupling geophysical models
  • BathymetryROMS a set of matlab scripts for handling bathymetry in ROMS
  • FEM_to_FD a script for getting a finite element mesh from a finite difference mesh

Programs for chemical applications

  • ChemicalReact a program for computing chemical equilibrium

Drawing programs

  • TorusDraw a program for drawing toroidal maps based on primal dual circle packings
  • CirclePack a program for drawing spherical maps based on primal dual circle packings

Programs for manipulating programs

  • MatlabScript a set of script for handling matlab Mfiles
  • LatexScript a set of script for handling papers written in latex

One purpose programs

  • DelaunaySix a program for enumerating all six-dimensional Delaunay polytopes
  • EXT-HYP7 enumeration of extreme rays of the hypermetric cone on seven vertices
  • PackingProbability.tar.gz a set of programs for dealing with continuous torus cube packings
  • Root matroid a set of programs for computing automorphism groups of root systems

Result of specific investigations

  • Perfect form in dimension 8 the list of them from the polyhedral computation
  • SpecialCuts the list of orbits of special cuts of 600-cell with the program used for their enumeration.
  • PeriodicTriangulations Data on periodic triangulations of n-dimensional lattice.
  • CutPolytopes the list of orbits of facets of the cut polytopes of some graphs.
  • ConvCoxeter the list of orbits of facets of some Coxeter groups.
  • PerfectCones the list of low rank faces of the perfect cone tesselation.
  • Additional information on the investigations of the book Geometry of Chemical Graphs: Polycycles and Two-faced Maps
  • Cube tilings and packings some programs and data set used for their enumeration.
  • Rigid lattices some Gram matrices of about 20000 rigid lattices in dimension 6.
  • SpaceFullerenes some data for space fullerenes.

Information available elsewhere

  • Point Groups a description of the point groups of 3 dimensional polytopes and plane graphs
  • Regular posets a description of the regular, semiregular, regular faced and Archimedean polytopes
  • The Leech lattice has 307 orbits of Delaunay polytopes, information at DelaunayLeech.tar.gz
  • Uniform partition of 3-space the list of 28 partition of Euclidean space by vertex transitive archimedean polyhedra

Data on metric cones

  • Metric and cut cones
  • Oriented semi-metric and Oriented Multicut cone
  • m-hemi-metric
  • (m,s)-super-metric

Delaunay tesselations of lattices

  • Delaunay polytopes of 14 out of 23 Niemeier lattices: Mendeley data set
  • DelaunayO23.tar.gz O23 (The Shorter Leech lattice) has 5 orbits of Delaunay polytopes
  • DelaunayLambda23.tar.gz The lattice Lambda23 has 709 orbits of Delaunay polytopes
  • DelaunayCutLattice.tar.gz Delaunay tesselation of some cut lattices

Contact polytopes of lattices

  • ContactPolytopeLeech.tar.gz the contact polytope of the Leech lattice has 232 orbits of vertices
