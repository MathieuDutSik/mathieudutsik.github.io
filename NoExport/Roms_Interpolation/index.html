Roms Interpolation

  • The ROMS model is a finite difference oceanographic model.
  • The program does interpolation:
    • From a ROMS grid to another ROMS grid
    • From a ROMS grid to a set of ADCP/CTD locations
    • From a ROMS grid to the boundary of another ROMS grid (for nested applications)
  • Their main feature are;
    • Specific methods for 2D (free surface, sea surface temperature, ...), 2Duv (barotropic currents, surface currents, ...), 3D (temperature, salinity, ...), 3Duv (momentum u/v).
    • An initial computation is done and thereafter all computations are based on it and so very fast.
    • The use is (in principle) relatively easy.
