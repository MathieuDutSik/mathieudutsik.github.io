Dual description of some cut polytopes

  • We have computed the dual description of some cut polytopes by using the polyhedral software.
  • Data sets are available as GAP save file, which can be read by doing ReadAsFunction(File)() with File one of the files of the archive.
  • For K144, which is actually the correlation polytope of K44, data set is available at DataCOR44 or alternatively as DataK144
